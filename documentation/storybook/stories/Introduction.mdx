## ğŸš€ Quick Start

### Installation

```bash
# React with Tailwind CSS
npm install @mycomponents/react-tailwind

# React unstyled (headless)
npm install @mycomponents/react-unstyled

# React Native  
npm install @mycomponents/react-native
```

### Basic Usage

```tsx
import { SpeechToText, TextToSpeech } from '@mycomponents/react-tailwind';

function App() {
  return (
    <>
      <SpeechToText 
        onTranscript={(text) => console.log(text)} 
        continuous={true}
        lang="en-US"
      />
      <TextToSpeech 
        text="Hello, world!" 
        autoSpeak={true}
        showControls={true}
      />
    </>
  );
}
```
