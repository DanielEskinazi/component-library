# Quick Start

Build your first app with @mycomponents in minutes.

## Prerequisites

- Node.js 18+ and npm 9+
- Basic knowledge of React, Vue, or your chosen framework
- A code editor (we recommend VS Code)

## Create a New Project

<Tabs>
  <TabItem value="react" label="React" default>

### 1. Create a React App

```bash
npm create vite@latest my-app -- --template react-ts
cd my-app
npm install
```

### 2. Install Components

```bash
npm install @mycomponents/react-tailwind
```

### 3. Add Tailwind CSS

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Update `tailwind.config.js`:

```js
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@mycomponents/react-tailwind/**/*.js",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

Add to your `src/index.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### 4. Use Components

```tsx title="src/App.tsx"
import { useState } from 'react';
import { 
  SpeechToText, 
  TextToSpeech, 
  AudioRecorder,
  CopyToClipboard,
  ThemeToggle,
  ShareButton
} from '@mycomponents/react-tailwind';

function App() {
  const [transcript, setTranscript] = useState('');

  return (
    <div className="min-h-screen bg-gray-50 dark:bg-gray-900 p-8">
      <div className="max-w-4xl mx-auto space-y-8">
        <div className="flex justify-between items-center">
          <h1 className="text-3xl font-bold">My Components Demo</h1>
          <ThemeToggle />
        </div>

        <div className="bg-white dark:bg-gray-800 p-6 rounded-lg shadow">
          <h2 className="text-xl font-semibold mb-4">Speech to Text</h2>
          <SpeechToText 
            onTranscript={setTranscript}
            continuous={true}
          />
        </div>

        {transcript && (
          <div className="bg-white dark:bg-gray-800 p-6 rounded-lg shadow">
            <h2 className="text-xl font-semibold mb-4">Text to Speech</h2>
            <TextToSpeech text={transcript} />
            <div className="mt-4 flex gap-2">
              <CopyToClipboard text={transcript} />
              <ShareButton 
                title="Check out my transcript"
                text={transcript}
              />
            </div>
          </div>
        )}

        <div className="bg-white dark:bg-gray-800 p-6 rounded-lg shadow">
          <h2 className="text-xl font-semibold mb-4">Audio Recorder</h2>
          <AudioRecorder 
            onRecordingComplete={(audio) => console.log(audio)}
          />
        </div>
      </div>
    </div>
  );
}

export default App;
```

### 5. Run Your App

```bash
npm run dev
```

## Common Patterns

### Handling Permissions

Some components require user permissions (microphone, clipboard):

```tsx
import { useState, useEffect } from 'react';
import { SpeechToText } from '@mycomponents/react-tailwind';

function App() {
  const [hasPermission, setHasPermission] = useState(false);

  const requestPermission = async () => {
    try {
      const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
      stream.getTracks().forEach(track => track.stop());
      setHasPermission(true);
    } catch (error) {
      console.error('Permission denied');
    }
  };

  return (
    <div>
      {!hasPermission ? (
        <button onClick={requestPermission}>
          Enable Microphone Access
        </button>
      ) : (
        <SpeechToText />
      )}
    </div>
  );
}
```

### Styling Components

All Tailwind variants accept custom classes:

```tsx
<SpeechToText 
  className="bg-gradient-to-r from-purple-500 to-pink-500 p-8 rounded-2xl"
/>
```

### Error Handling

```tsx
<SpeechToText 
  onError={(error) => {
    console.error('Speech recognition error:', error);
    alert('Please check your microphone permissions');
  }}
/>
```
